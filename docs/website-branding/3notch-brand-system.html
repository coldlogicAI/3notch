01 — Brand Concept
What 3Notch Is

3Notch is the format and the local plumbing for a hand-off you direct but no longer have to carry by hand.

A local-first CLI and MCP server for moving AI project context across boundaries — between tools, repos, projects, and machines. You're already the bus for your AI context, copying summaries between Claude, Codex, Cursor, ChatGPT. 3Notch is the packet layer you wish you'd had. The bytes move between tools; you stay the author and reviewer.

Local-first Vendor-neutral Reviewable
Brand Position
PACKETS not memory
Built for cross-boundary transfer
Tools change · context shouldn't
Origin. Travelers carved three horizontal notches into roadside trees to signal a way through. The marks stayed long after the traveler moved on. A packet is a notch. An inbox is a trail. The product is what makes the trail readable to anyone who comes next — another tool, another agent, another person, or you in six months. Keep this origin brief everywhere it appears.

02.5 — Alternate Marks
A — Inline Mark (text-row)
Wider aspect ratio for use beside a wordmark on a single text-line, or as a section divider inside long-form copy. Same stroke palette; squeezed vertically.
B — Monochrome (print / 1-color)
For black-only print, embroidery, single-color embossing. Mark color drops; geometry stays. The primitive is recognizable without the red.
C — Inverse (mark on red)
Sticker, swag, social-promo cards. Paper-on-mark when the canvas itself should carry the brand color. Use sparingly — the canonical mark is mark-on-ink.

Mark Red — One Accent, Period
The mark red is the only chromatic accent in the system. It carries: the icon mark, the wordmark numeral, the eyebrow labels, the active-store dot, the primary button, the inline emphasis, the hover state. Every accent shares one color. Restraint is the personality.
Status States Without a Second Accent
Success = paper text with a mark-red ✓ glyph. Warning = paper-dim text with weight. Error = paper text with a mark-red prefix or border-left. The system does not introduce green/yellow/blue for semantic state — the glyph and the weight carry the meaning, not a new hue. (Earlier v0.1 included a Signal green; removed in this revision.)
Mark red reads as ember, signal-flare, trail-blaze — warm and intentional, never corporate. Paper-on-ink is the foundation. No bark, moss, sky, or gold ever enter the palette — those colors live on the mascot's body and his world, never in the brand system.

The number 3 is semantically tied to the mark (three notches) and to the trail-blazing origin. It stays. Three written forms operate across three contexts: display (3Notch), CLI (notch), package (@3notch). Enforce per-context; do not mix.
Voice — Direct, technical, quietly confident
Sounds like
"Mark your trail."
"A packet, an inbox, a way through."
"Your AI tools will change. Your project context shouldn't have to."
"Three horizontal marks. That's the whole idea."
Does not sound like
"Revolutionize your AI workflow!"
"The substrate for agentic workflows that compound."
"Empowering teams to unlock context."
"Built different, ship faster."
"Howdy, partner — let's blaze a trail."
Writes like a thoughtful developer tool. Short sentences. Low ceremony. No marketing puff. No exclamation marks. No "we" referring to a non-existent team. Earns moments of warmth through restraint everywhere else.
Tagline — Primary Only
Mark your trail.
"Mark your trail." Three words, present tense, action verb. The marks the user leaves are notches — the tagline implicitly explains the name without forcing the noun into a verb. Use the README line ("Your AI tools will change. Your project context shouldn't have to.") as the supporting paragraph under the hero, never as the tagline itself. Retired alternates: "Leave a notch. Mark your trail.", "Package the session. Send it anywhere.", "The substrate for agentic workflows that compound."

REMOVED · drop shadow / glow (noise at small sizes)
REMOVED · gradient stroke (single color is the discipline)
REMOVED · second mark color / outline (one color, period)
All icon marks at any size derive from this base. The 80×80 viewBox is canonical; render larger sizes by scaling proportionally and reducing stroke-width gain. The mark must look identical when rasterized to favicon, displayed at 44px in nav, or printed at 4 inches on a sticker.
10 — Mascot Usage Rules
Under Evaluation
Current Placeholder Mascot
Status
AI-rendered lumberjack — Pixar-style 3D character, red plaid flannel, red beanie, axe over shoulder, standing beside a tree carrying three clean horizontal trail-blaze notches. Acts as the current personality placeholder while a replacement concept (a culturally-neutral trail-scout with satchel-of-packets + carving tool) is being generated and evaluated. Surface restrictions below apply regardless of which mascot is final.
Role
Personality, not theme. Never drives the layout or palette. Bark, moss, sky, gold, and the saturated flannel live on the character's body and his world — they do not enter the brand palette. The mark and the wordmark always lead; the mascot supports.

The mascot is brand warmth; the three-line mark is brand identity. Identity leads on every surface a stranger sees first. Warmth lives on surfaces where they've already opted in (clicked through a 404, read release notes, found the easter egg, bought a sticker). If a surface would be weaker with a clean abstract treatment than with the mascot, the mascot doesn't belong there.
11 — Usage Rules
✓ Always
✓ Reproduce the mark from canonical SVG (Section 09)
✓ Write 3Notch with capital T-N, no space
✓ Use Geist 600 for the wordmark
✓ Use Geist Mono for CLI / labels / data
✓ Use Instrument Serif italic for one accent / page
✓ Mark red as the only chromatic accent
✓ Color the numeral 3 mark-red inside the wordmark
✓ Scale the mark proportionally with stroke-width gain
✗ Never
✗ Add a container box, glow, or gradient to the mark
✗ Render the wordmark in Geist Mono or Instrument Serif
✗ Introduce a second accent color (green, blue, yellow)
✗ Use exclamation marks in product copy
✗ Write "we" implying a team that doesn't exist
✗ Stretch, rotate, or skew the mark
✗ Color any letter other than the 3 inside the wordmark
✗ Set body copy in Instrument Serif
The brand's credibility comes from precision and restraint — not volume. Every element should feel like a developer tool, not a SaaS landing-page template. If a contributor's PR introduces a second accent color, a fourth typeface, an exclamation mark, or a stretched mark — reject and link this section.
